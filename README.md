# Code Line Counter
代码行数统计工具

## Simple Usage
python code_line_counter.py  [path]

## full Usage
python code_line_counter.py [-h] [-e EXCLUDE [EXCLUDE ...]] [-i INCLUDE [INCLUDE ...]] [path]  

positional arguments:  
  path                  要统计的目录路径，默认为当前目录  

options:  
  -h, --help            show this help message and exit  
  -e EXCLUDE [EXCLUDE ...], --exclude EXCLUDE [EXCLUDE ...]   排除的目录名称    
                       
  -i INCLUDE [INCLUDE ...], --include INCLUDE [INCLUDE ...]   只包含指定的文件扩展名
